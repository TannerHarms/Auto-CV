# Projects

## [Auto CV](https://github.com/TannerHarms/Auto-CV)

Markdown-to-resume builder that converts an Obsidian-style vault into
LaTeX/PDF, DOCX, and HTML resumes with customisable style presets.

**Technologies:** Python, Pydantic, Jinja2, Typer

## [StreamWatch](https://github.com/jrivera-dev/streamwatch)

Real-time monitoring dashboard for Kafka consumer lag with alerting
via Slack and PagerDuty integrations.

**Technologies:** Go, React, Kafka, Prometheus

## DevDocs Aggregator

CLI tool that aggregates API documentation from multiple sources into
a unified, searchable local index.

**Technologies:** Python, SQLite, Click
