## Selected Work

### CloudScale Event Platform

Designed and built a horizontally scalable event processing platform handling
2M+ events per day. The system uses Kafka for ingestion, Python workers for
processing, and Kubernetes for orchestration.

**Key achievements:**
- Sub-second end-to-end latency at the 99th percentile
- Zero-downtime deployments via blue-green strategy
- Automated scaling based on consumer lag metrics

### Fraud Detection Pipeline

Built a real-time fraud scoring service that evaluates transactions in under
100ms. The model combines rule-based heuristics with an ML scoring layer
and integrates directly with the payment gateway's webhook flow.
